IPCC Bridge Authors

This script loads the file bridge_authors_by_ar_wg_chapter.csv and builds a GEXF file where the positions of chapters in each AR are pre-computed.

Warning: your browser may prevent this page from downloading multiple files. Fetch its warning message to allow multiple downlads on this page

This script will download 7 files:

  • Bridges for AR 1.gexf to Bridges for AR 5.gexf: For each AR, authors participating to different working groups in this AR
  • Bridges Intra AR.gexf: Authors participating to different working groups in the same AR. Data across all ARs.
  • Bridges Inter AR.gexf: Authors participating to different working groups in different AR. Data across all ARs.
